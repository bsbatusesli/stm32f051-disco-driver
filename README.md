# Bare-Metal Driver development for STM32F051xx Microcontroller
## Developing microcontroller drivers from stratch for educational purposes

This repo contains driver files and application files written for stm32f051 discovery board. Repo is still under the construction. In MCU specific header file can be used in other projects or can be used as a reference. Drivers can have some bugs.

Status of included drivers:
* GPIO (Done)
* SPI (On going)
* I2C (Not yet)
* UART (Not yet)


 [MCU1](https://www.udemy.com/course/mastering-microcontroller-with-peripheral-driver-development) Udemy course given by FastBit Brain Academy is followed in development process.
